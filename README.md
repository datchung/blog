# blog
Personal blog

https://datchung.github.io/blog

# Getting Started

1. Run `npm install`
2. Run `npm run start`
3. Open http://localhost:3000 in a browser

# Features

* File watcher with lint and build
* Hot reloading
* Optimized production/release build
* String localization in CN, EN, ES, FR, PT, RU
* Unit tests (TODO)
